# 요트 다이스 멀티 플레이

## 주요 기능

### 주사위 기능

- 5개의 주사위가 존재한다.
- 주사위를 굴리면 1 ~ 6 사이 랜덤 값을 가진다.
- 특정 주사위를 굴리지 않도록 고정할 수 있다.
- 고정된 주사위를 해제할 수 있다.

### 점수 계산 기능

- 주사위 결과에 대해 각 족보별 점수를 계산한다.
- 소계(에이스 ~ 식스 점수 합계)를 계산한다.
- 소계가 63점 이상이면 보너스 점수를 얻는다.
- 총합 득점(모든 족보 점수 + 보너스 점수)을 계산한다.

### 게임 방 기능

- 두 명의 플레이어와 하나의 주사위 세트를 가진다.
- 현재 플레이어는 주사위 굴리기 또는 고정/해제, 점수 기록을 할 수 있다.
- 현재 주사위 세트로 가능한 족보별 점수를 계산한다.

### 게임 통신 기능

- 웹 소켓을 통해 실제 플레이어(클라이언트)와 통신한다.
- 유효한 게임 방에 대한 요청인지 확인한다.
- 현재 플레이어의 턴인지 확인한다.
- 현재 게임 방의 상태를 응답한다.

### 게스트로 참가 기능

- 클라이언트로부터 닉네임을 받아 고유한 아이디로 구분되는 플레이어를 생성한다.
- 생성된 플레이어 정보(아이디와 닉네임)를 응답한다.

### 무작위 매칭 기능

- 매칭 대기열에 플레이어를 등록한다.
- 대기열에 두 명 이상이 모이면 즉시 매칭을 성사시키고 게임 방을 생성한다.
- 매칭된 플레이어에게 웹 소켓을 통해 매칭 성사 메시지를 전달한다.
- 대기 중인 플레이어는 매칭을 취소할 수 있다.

### 우승자 판단 기능

- 게임이 종료되면 최종 점수가 가장 높은 플레이어를 우승자로 판단한다.
- 가장 높은 최종 점수를 가진 플레이어가 2명 이상일 경우 무승부로 판단한다.
